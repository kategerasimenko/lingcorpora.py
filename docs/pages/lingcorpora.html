Lingcorpora

This package includes API for National Corpus of Russian Language Parallel subcorpus of National Corpus of Russian Language Center of Chinese Linguistics corpus Corpus Bambara de Reference * Maninka Automatically Parsed corpus

R version of this package by George Moroz is located here.

The package is available under MIT license.

About

The project is maintained by George Moroz and several students of School of Linguistics of Higher School of Economics. At different stages, the following people have contributed to the project: Alexey Koshevoy, Mark Sobolev, Ekaterina Gerasimenko, Anna Zueva, Diana Malyshok, Artyom Kopetskiy, Maria Terekhina, Ustinya Kosheleva.

Contributing

Reporting a bug & requesting functionality

You can report a bug, ask a question or suggest adding features via Issues in the repository.

Making new corpora and proposing changes

You are welcome to suggest your improvements to the source code and make APIs for more corpora. You should propose changes via pull requests. Your code should fit into the overall structure, which is described in the guide.
